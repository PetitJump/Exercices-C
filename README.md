# Description
Evaluateur de Note en donnant un commentaire basique en fonction de la note.
Il demande a l'utilisateur si il veut remettre une note ou non (utilisation d'une boucle).
Projet qui a pour but d'apprendre les bases du C
Premier projet en C

# Fonctionalité aprise
- Apprendre à utiliser les variables et types en C (int, char[]).
- Comprendre la lecture de l’utilisateur avec scanf.
- Manipuler les conditions (if, else if, else).
- Introduire les fonctions et leur retour de valeur (char* demander_prenom()).
- Utiliser les boucles pour répéter une action.
- Découvrir la gestion des chaînes de caractères avec char[] et strcmp.
